_Iframe-resizer_ ships official packages for every major front-end framework,
as well as a framework-agnostic Web Component and a jQuery plugin. Every
package is written in TypeScript and ships with first-class type
definitions.

- [Alpine.js](/frameworks/alpine) — `@iframe-resizer/alpine`
- [Angular](/frameworks/angular) — `@iframe-resizer/angular`
- [Astro](/frameworks/astro) — `@iframe-resizer/astro`
- [jQuery](/frameworks/jquery) — `@iframe-resizer/jquery`
- [React](/frameworks/react) — `@iframe-resizer/react`
- [Solid](/frameworks/solid) — `@iframe-resizer/solid`
- [Svelte](/frameworks/svelte) — `@iframe-resizer/svelte`
- [Vue](/frameworks/vue3) — `@iframe-resizer/vue`
- [Web Component](/frameworks/web-component) — `@iframe-resizer/web-component`

There is also a [Framework API](/api/framework) that provides a simplified
interface for creating your own _iframe-resizer_ integration for any other
framework or library.

If you create a wrapper component or library plugin for _iframe-resizer_,
or include it in another component library, please open a
[PR](https://github.com/iframe-resizer/docs/pulls) to add it to this page.
